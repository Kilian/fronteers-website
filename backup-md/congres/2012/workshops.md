     <h2>Fronteers 2012 workshops</h2>
     <p>Just like previous Fronteers editions we'll be hosting two workshops prior to the conference. Unlike previous years both workshops will take place on the same day: Wednesday, 3 October.</p>
     <h3 id="javascript-application-development"><a href="/congres/2012/workshops/javascript-web-apps-tooling-addy-osmani">JavaScript Web Apps &amp; Tooling  by Addy Osmani</a></h3>
     <p>Learn how to build highly maintainable web applications using modern JavaScript frameworks and a polished tooling workflow. <a href="http://addyosmani.com/">Addy</a> will take you through how to refactor your applications so that they’re more maintainable, taking you through modern architectural design patterns, frameworks and tools that can help make writing an app a joy again. <a href="/congres/2012/workshops/javascript-web-apps-tooling-addy-osmani">More information</a>.</p>
     <h3 id="visual-design-101"><a href="/congres/2012/workshops/visual-design-101-for-web-developers-mark-boulton">Visual Design 101 for Web Developers by Mark Boulton</a></h3>
     <p>Visual design is not magic, or the result of some God-given talent. Anyone can achieve it by applying rules, techniques and practice.<br><a href="http://www.markboultondesign.com/">Mark</a>’s workshop will cover the basics of visual design. From collaborative idea generation and visualising ideas, to the fundamentals of typography, grid design, composition and colour theory. <a href="/congres/2012/workshops/visual-design-101-for-web-developers-mark-boulton">More information</a>.</p>
     <h3>Venue</h3>
     <p><a href="http://www.felix.meritis.nl/en/">Felix Meritis</a><br>Keizersgracht 324<br>1016 EZ Amsterdam</p>
